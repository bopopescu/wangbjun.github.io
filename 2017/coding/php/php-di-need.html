首先，先看看一句大胆的言论：“大多数情况下，你不需要一个依赖注入容器也能受益于依赖注入”

容器包装了需要实现的基本特性，依赖注入容器管理对象：从初始化到配置。对象本身不知道自己被容器管理，更不知道容器。所以容器可以管理任何PHP对象，如果对象使用依赖注入解决依赖就更好了，但那不是先决条件。
